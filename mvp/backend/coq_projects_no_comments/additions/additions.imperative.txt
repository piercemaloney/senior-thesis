

Set Asymmetric Patterns.



Goal

forall (A : Set) (Pre Post : A -> Prop) (B : Set),

{a : A | Pre a} ->

(forall a : A, Pre a -> {a' : A | Post a'}) ->

(forall a : A, Post a -> B) -> B.

exact

 (fun A Pre Post B start body return_ =>

  match start with

  | exist a pre =>

      match body a pre with

      | exist a' post => return_ a' post

      end

  end).

Save Imperative.
